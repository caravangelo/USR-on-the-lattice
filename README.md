# Ultra-Slow-Roll on the Lattice: Animations

This repository contains animations of the simulations presented in [arXiv:2410.23942](http://arxiv.org/abs/2410.23942), [arXiv:2506.11795](http://arxiv.org/abs/2506.11795), and [arXiv:2504.?????](http://arxiv.org/abs/2604.?????).

**These simulations were performed using [InflationEasy](https://github.com/caravangelo/inflation-easy.git), an open-source lattice code for inflation.**

The animations cover two main categories of models:

- **Mild non-Gaussianity (NG) models**, with three subcases (Case I, Case II, Case III of [arXiv:2410.23942](http://arxiv.org/abs/2410.23942)), each shown as a separate animation.  
- **A large NG case**, studied in [arXiv:2504.?????](http://arxiv.org/abs/2504.?????).

In each animation:

- The **left panel** shows 2D snapshots of the inflaton field fluctuations $\delta\phi$ as a function of e-fold time $N$, normalized by the standard deviation $\sqrt{\langle \delta\phi^2 \rangle}$.  
- The **right panel** displays the 1-point probability density function (PDF) of inflaton field values across the lattice, overlaid with the potential (green curve). The red line indicates the lattice average, while the dashed line shows the prediction from standard perturbation theory (SPT).

These simulations correspond to potentials for which SPT predicts a maximum curvature power spectrum $\mathcal{P}_{\zeta,\mathrm{tree}}^{\mathrm{max}} = 1$ in all cases.

For definitions and notation, please refer to the papers.
